    # 我的第一个博客

    在还没有接触网络编程的时候，我就想过建立博客。至于目的嘛，我并不清楚，不过原因我知道，只是跟风，因为几乎所有的同学都有自己的博客。

    市面有wordpress/hexo等等大量的产品可以选择，但是让没有任何编程经验的我来看它们的文档，我只能说感觉云里雾里的。不是因为这些文档描述不清，而是因为我一直再自我提醒着“你还没有学过任何编程知识，使用这些成熟产品可以做出看起来很漂亮的东西，没有任何值得自信的地方”。经过多次考虑，我决定先学会基本的编程知识，再来建立博客。

    于是利用空闲时间，我断断续续学习了web前端。今天，我终于有勇气使用市面上成熟的产品，来搭建博客了。🎉

    # 我做过的傻事

    在读完菜鸟教程 (opens new window)里的html/css/js入门文档后，结合看到的如何在GithubPages (opens new window)建立静态博客的文章，我想我完全可以不用Hexo/Hugo，从零开始写博客的页面。使用MDUI (opens new window)框架，同时模仿着别人的博客，我花了接近两个月的时间做出来完全写死的网页。最后我知道，每次写文章都要在各个已有的页面里手动添加文章，后期开发和维护会十分麻烦且龟速。

    在上面提到的那段时间里我又有了学习Vue.js (opens new window)的想法，而学习Vue没学过Node.js (opens new window)还有Webpack (opens new window)是不行的。所以我傻乎乎得花了一个月读完Vue/Nodejs/Webpack的官方文档，随后抛弃jQuery (opens new window)用这几年流行的技术栈，重写了之前那个完全不具有价值的仿博客页面。

    经过实践，我知道，网络编程只会前端不会后端是不行的，所以我用6个月的时间学习了golang和一些计算机专业的核心基础课程，根据框架Iris (opens new window)的文档写了一个小demo。

    今天，在对网络编程有一定了解以后，从“必然王国”接近“自由王国”的情况下，我终于能自由轻松地、没有心理负担地使用VuePress (opens new window)搭建博客了。😜

    # 博客用来做什么

    我暂时能想到的暂时只有这几点吧：

    1. 记录生活，比如：发现的问题、对问题的思考；
    2. 读书笔记，emmm，这个可以有（小声告诉所有人，我读书不多🤣）；
    3. 工具知识，像网络编程、电子技术等等的吧（写一些大佬们不屑于去记录的白痴知识）。

    # 结束语

    昨天是世界末日，明天也是，今天是不是呢？

    想说的说完了，就这样吧。
