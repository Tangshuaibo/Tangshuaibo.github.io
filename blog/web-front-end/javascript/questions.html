# 9. 理解观察者模式与发布订阅模式

参考文章：

  • js - 观察者模式与订阅发布模式
  • JavaScript 设计模式（六）：观察者模式与发布订阅模式
  • DOM事件监听的实现基于发布订阅模式。
  • 观察者模式：需要被观察对象维护观察者列表。
  • 发布订阅模式：事件中心不维护订阅者列表，而维护一系列事件列表与事件触发的动作列表。
  • 即区别在于：一个维护的是观察者队列，另一个维护的是事件触发的动作队列。
